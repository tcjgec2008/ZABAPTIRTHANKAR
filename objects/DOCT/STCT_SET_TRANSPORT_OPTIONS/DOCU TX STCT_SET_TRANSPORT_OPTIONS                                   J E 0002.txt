U1 &PURPOSE&                                                               
*  このタスクは、有効化する OData サービスの移送オプションの設定に使用しま                                 
   す。                                                                      
U1 &CHECK&                                                                 
*  このタスクでは、トランザクション <DS:TRAN.SCC4>SCC4</> の設定でクライア                         
   ント非依存の変更と変更の自動記録が許可されているかどうかをチェックしま                                     
/  す。                                                                      
U1 &REALIZE&                                                               
*  OData サービスを登録する接頭辞とパッケージを入力します。                                         
*  デフォルトでは、サービスはパッケージ $tmp に接頭辞 <ZK>Z</> を付けて登録さ                           
/  れます。                                                                    
*  移送可能パッケージを選択する場合は、ワークベンチ依頼とカスタマイジング                                     
/  依頼が必要になります。                                                             
*  ワークベンチ依頼は OData サービスと ICF サービスのデータを記録し、カス                               
/  タマイジング依頼は OData サービスへのシステムエイリアスの割当を記録しま                                 
/  す。                                                                      
*  新しい依頼とタスクを自分のユーザに登録するか、または既存の依頼を選択す                                     
/  ることができます。                                                               
*  非依存/依存の変更と変更の自動記録が許可されていない場合は、クライアント                                    
/  の設定を変更するためのチェックボックスをオンにしてください。
